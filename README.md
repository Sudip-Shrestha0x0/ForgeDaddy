# ForgeDaddy

Free online tool suite — 70+ browser-based tools for PDF, video, image, resume, coding, and more.

[Use Tools](https://forgedaddy.com)

All tools run 100% in your browser. 

Everything runs client-side, meaning:

- No uploads (Your data stays on your device)
- No signups
- No paywalls
- No watermarks

## Tools Include

- **PDF:** Merge, Split, Compress, Convert to Word/Image, OCR
- **Video:** Compress, Convert to MP3, Screen Recorder
- **Image:** Background Remover, Upscaler, SVG Converter, Compressor, Resizer
- **Resume:** ATS Resume Builder (PDF + DOCX), Resume Checker, AI Detector
- **Finance:** Mortgage Calculator, Invoice Generator
- **Legal:** NDA Generator, Privacy Policy Generator
- **Text:** Word Counter, Markdown Preview, Fancy Text, Citation Generator
- **Dev:** JSON Formatter, Regex Tester, Base64, JWT Decoder, Diff Checker
- **Design:** Gradient Generator, Box Shadow, Color Palette
- **And 30+ more**

## Privacy

Your files and data never leave your device. Everything is processed client-side using JavaScript. No servers, no APIs, no tracking of your content.
